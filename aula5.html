Estrutura de Repetição

Existem situações onde uma mesma ação ou evento ocorre diversas vezes. Um exemplo: Suponhamos que você está fazendo uma receita de bolo que usa cinco ovos no preparo, então você irá executar a ação de quebrar os ovos diversas vezes, nesse caso da receita essa ação seria repetida cinco vezes que é o número necessário de ovos que a receita pede.

Esse é um problema comum em programação, ter que repetir uma sequência de ações inúmeras vezes de forma repetida e sequencial. De forma simples, seria possível apenas sair replicando o código a quantidade de vezes desejadas. Porém isso não é muito interessante, certo? Para lidar com isso é usado uma estrutura de repetição.

As estruturas de repetição são blocos onde uma sentença é verificada e se verdadeira um bloco de código é executado, uma dessas estruturas de repetição é o “para” ou, como é usado em inglês, FOR.

Usando como exemplo a ideia da receita como sendo um algoritmo, para conseguir colocar todos os ovos necessários seria possível replicar o código de “quebrar o ovo” quantas vezes fosse necessário. Se dessa receita for feita apenas uma medida seria fácil fazer isso, porém se a quantidade de ovos fosse muito maior teríamos que repetir esse mesmo código muitas vezes e isso não seria eficiente.

For

O for é uma estrutura que irá repetir um código uma quantidade de vezes até uma condição ser verdadeira Sua estrutura básica é composta por:

  1. Uma variável de controle: usada para saber quantas vezes o código já foi executado e talvez comparar com a sentença.;
  2. Uma condição de parada;
  3. E as instruções que deverão ser executadas.

Ao executar o comando for, primeiramente é testada a condição, se ela for verdadeira o comando for se encerra e passa para próxima instrução, caso a condição de parada seja falsa o código dentro do bloco do comando for é executado e a variável de controle é modificada conforme foi programada.

Em resumo, com o for é possível executar blocos de comandos quantas vezes forem necessários e utilizando uma contagem simples de vezes para controlar.
